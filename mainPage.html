Identifique la primera palabra y la ultima de cada bloque con las pistas que se dan. A continuacion, trate de descrubir las palabras intermedias. Para lograrlo, cambie una letra de la primera palabra para obtener la proxima y, despues, altere el orden de una o varias letras para encontrar la siguiente. Siga asi, sucesivamente, hasta que logre completar ambos casilleros. Todas las palbabras intermedias deben tener significado.
